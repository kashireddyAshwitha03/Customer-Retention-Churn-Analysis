# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer subscription data to identify churn patterns, retention trends, and customer lifetime value (CLV). The goal is to provide actionable business insights for improving customer retention.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Churn Analysis
- Retention Analysis
- Customer Lifetime Value (CLV)
- Cohort Analysis
- Business Recommendations
